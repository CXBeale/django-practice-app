## MVP Feature List

### 1. User Accounts & Authentication
- Users can register an account
- Users can log in and log out
- Each user can only access their own food entries

---

### 2. Food Entry Management

#### Create Food Entries
- Users can add a food entry
- Required fields:
  - Food name
  - Quantity
  - Date
- Optional notes field

#### View Food Log
- Users can view a list of their food entries
- Entries are displayed by date (most recent first)

#### Edit Food Entries
- Users can edit an existing food entry

#### Delete Food Entries
- Users can delete an existing food entry

---

### 3. Dashboard Overview
- Users can view a dashboard
- The dashboard displays:
  - Number of food entries logged today
  - Recent food entries (optional)

---

### 4. Navigation & Basic UI
- Navigation bar is available on authenticated pages
- Navigation includes links to:
  - Dashboard
  - Food Log
  - Add Food Entry
  - Logout

---

### Out of Scope for MVP
- Nutrition analysis (calories, macros, etc.)
- Activity tracking
- Charts or visual analytics
- External food databases
- Mobile optimization
