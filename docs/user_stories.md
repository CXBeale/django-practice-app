## User Stories (Food Tracking MVP)

### Account & Access
- As a user, I want to create an account so that my food entries are saved and private.

### Food Logging
- As a user, I want to add a food entry so that I can record what I have eaten.
- As a user, I want to include a food name and quantity so that my entries are meaningful.
- As a user, I want food entries to be associated with a date so that I can track what I ate on specific days.

### Viewing & Managing Entries
- As a user, I want to see a list of my food entries so that I can review my eating history.
- As a user, I want to edit a food entry so that I can correct mistakes.
- As a user, I want to delete a food entry so that I can remove incorrect or unwanted entries.

### Dashboard & Navigation
- As a user, I want to see a simple summary of today’s food entries so that I know what I’ve logged today.
- As a user, I want to easily navigate between my dashboard and food log so that logging food is quick.
